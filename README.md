✨ KEY FEATURES
================================================================================

- 🔐 Secure Authentication (JWT / OAuth ready)
- 👥 Role-based Access Control (Admin / Donor / Organization)
- 💰 One-time & Recurring Donations
- 📊 Real-time Analytics & Impact Tracking
- 🎯 Campaign Management System
- 🌙 Dark / Light Mode Support
- 📱 Fully Responsive Design

================================================================================
🧠 PROBLEM STATEMENT
================================================================================

Many charity platforms suffer from:
- Lack of transparency in donations
- Poor user experience
- No real-time tracking of impact
- Weak campaign management tools

💡 SOLUTION:
Ataa solves these issues by providing:
- Transparent donation tracking
- Real-time campaign progress
- Modern UX/UI experience
- Scalable role-based system

⚙️ INSTALLATION
================================================================================

git clone https://github.com/Sayed-Herzallah/Ataa-Charity-Platform-React-TypeScript.git

cd Ataa-Charity-Platform-React-TypeScript

npm install

npm run dev

================================================================================
🌍 ENVIRONMENT VARIABLES
================================================================================

VITE_API_BASE_URL=https://api.ataa-platform.com
VITE_API_TIMEOUT=10000

VITE_STRIPE_PUBLIC_KEY=your_key
VITE_PAYPAL_CLIENT_ID=your_id

VITE_DARK_MODE_DEFAULT=system
VITE_DEFAULT_LANGUAGE=en
